响应式原理

发表于 2022-12-18 更新于 2022-12-20 分类于 Vue2源码

目标：基于defineProperty，对 data 数据进行劫持，实现响应式

关键学习点：指针，闭包，递归，this指向（这应该包括在指针里吧）

技巧：…arg(作为参数) => []，会转换成数组; newObj = Object.create(obj)，obj作为“新创对象(newObj)”的原型

劫持 data

Vue2的问题：对整个data对象进行重写，性能差

Object.defineProperty 只能劫持已存在的属性（理解：新增的）

劫持 data 对象

在src下创建 observe -> index.js

关于劫持为什么只能是对象的原因：
若data里的属性作为一个值类型并处于第一层，那么它第一次创建实例就已经被劫持
而引用类型，只劫持当前的指针（也就是第一层），其对应的值（属性）并不会被劫持，所有才继续判断，需要结合后面才能理解这的写法
同时这里也为后面新增劫持作铺垫

劫持 data的属性 里的对象
